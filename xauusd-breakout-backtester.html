XAUUSD Breakout Backtester

Simulate a support/resistance breakout strategy on real daily gold (XAU/USD) prices, or upload your own MT5 trade history to check it against funded-account rules. Either way, results are checked against a daily loss limit, max drawdown, and profit target.

Setup

Synthetic data notice: only daily closes are available from the free data source, so sub-daily bars are generated by interpolating a random-walk bridge between each real daily close. The daily open/close points are real; everything in between is a statistically plausible but fabricated path. This is useful for sanity-checking your entry/exit logic at finer granularity, but the exact trade prices, whipsaws, and stop/target touches are not real market data — don't treat these results as validation of live intraday performance. Re-running the same window will generate a different random path each time.
In the MT5 terminal: open the History tab, right-click, choose Report → Save as Report (produces an .htm file covering the period shown). Upload that file here — your actual closed trades are read directly from it; no strategy is simulated. A .csv export with columns like Time, Symbol, Type, Volume, Price, Profit, and (ideally) Balance also works.

Data: freegoldapi.com, no key required — daily gold (XAU/USD) prices sourced from Yahoo Finance gold futures (GC=F) from 2025 onward; earlier dates fall back to monthly/annual sources and are too sparse for this backtest. Real prices are daily closes, not intrabar highs/lows. Choosing a 4H/1H/15m timeframe does not fetch real intraday data (no free, no-key, CORS-enabled source for that exists) — instead it fabricates plausible sub-daily bars between each pair of real daily closes via a seeded random-walk bridge, so you can sanity-check how your lookback/SL/TP logic behaves at finer granularity. Only the daily open/close anchor points are real. The rules check marks open positions to market at each bar's close to approximate loss since the start of each calendar day, and uses a trailing peak-to-equity drawdown — verify the exact calculation method (balance vs. equity, static vs. trailing) against your funded account's actual rules, since firms vary. Educational tool only — not investment or trading advice.
